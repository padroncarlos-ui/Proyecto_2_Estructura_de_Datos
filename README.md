# Proyecto_2_Estructura_de_Datos
Proyecto 2 de estructura de datos (Hash Tables y Árboles AVL)
Proyecto que simula una cola de impresión y que utiliza colas de prioridad para determinar que archivo se imprimirá primero
Al cargar un archivo csv con el formato indicado el usuario podrá elegir si imprimir un documento con prioridad para que se imprima de primero o dejarlo en la cola de espera si no es prioritario.
